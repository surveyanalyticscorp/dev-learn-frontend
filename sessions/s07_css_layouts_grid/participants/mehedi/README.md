# CSS Grid Layouts Workshop

## Overview

Complete the 11 CSS Grid tasks by implementing the required grid layouts. Each task focuses on different CSS Grid concepts and progressively increases in difficulty.

## Getting Started

1. Open `index.html` in your browser
2. Edit the `tasks.css` file to complete each task
3. Solutions are available in `solutions.css` (comment/uncomment the link in HTML)

## Tasks

### Task 1: Basic 3-Column Grid
Create a simple grid with 3 equal-width columns.

### Task 2: Adding Gaps
Create a 3-column grid with 20px column gap and 40px row gap.

### Task 3: Spanning Columns
Create a 3-column grid where the first item spans all 3 columns.

### Task 4: Grid Template Areas
Use `grid-template-areas` to create a sidebar-main layout with header and footer spanning both columns.

### Task 5: Mixed Units
Create a layout with fixed 150px side columns and fluid middle column.

### Task 6: Implicit Grid
Define a 3-column grid with auto-generated 100px tall rows.

### Task 7: Centering
Use a single grid property to center content both horizontally and vertically.

### Task 8: Responsive Cards
Create a responsive card gallery with minimum 250px column width.

### Task 9: Nested Grids
Implement a 2-column grid with nested 2-column grid inside the second item.

### Task 10: Holy Grail Layout
Create the classic layout with header, footer, navigation (120px), main content (fluid), and aside (120px).

### Task 11: Bento Box Layout
Create a complex 4x4 responsive layout using `grid-template-areas` that collapses to single column on mobile.

## Files

- `index.html` - HTML structure with all tasks
- `base.css` - Base styling (do not edit)
- `tasks.css` - Your implementation file
- `solutions.css` - Complete solutions

## Instructions

Work through each task in order. Use CSS Grid properties to achieve the specified layouts. Reference the task descriptions in the HTML for detailed requirements.
